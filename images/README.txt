FOREVER STILL BEAUTY — Images Folder
=====================================

Place your image files here and update the HTML to reference them.

RECOMMENDED IMAGE SIZES:
- Hero background:         1920 x 1080px (or 2560 x 1440px for retina)
- Brand story / about:     800 x 1000px (portrait orientation)
- Collection cards:        600 x 800px (portrait orientation)
- Product cards:           800 x 880px (nearly square, portrait)
- Founder photo:           600 x 800px (portrait)
- OG / social share image: 1200 x 630px (landscape)

HOW TO SWAP IN REAL IMAGES:
1. Add your image file to this folder.
2. Open the relevant HTML file (index.html, about.html, shop.html, contact.html).
3. Find the comment that says <!-- REPLACE: ... --> near the relevant section.
4. Replace the placeholder <div> with a real <img> tag. Example:

   BEFORE:
   <div class="product-card__img-fill" role="img" aria-label="...placeholder"></div>

   AFTER:
   <img src="images/product-glow-ritual-set.jpg"
        alt="The Glow Ritual Set — morning and evening skincare essentials"
        style="width:100%;height:100%;object-fit:cover;">

5. Always include descriptive alt text for accessibility and SEO.

FORMAT RECOMMENDATIONS:
- Use JPG for photographs (best file size for quality ratio)
- Use PNG for graphics with transparency
- Use WebP if you want better performance (add a JPG fallback)
- Compress images before uploading (tools: TinyPNG, Squoosh, ImageOptim)
